# Event Registration System

## Objective
Develop a backend application to manage event registrations.

## Technologies Used
- Python
- Flask
- Flask-SQLAlchemy
- SQLite

## Features
- User Registration
- Event Creation
- View Events
- Register for Event

## API Endpoints
- POST /users
- POST /events
- GET /events
- POST /register

## Conclusion
This project demonstrates backend API development using Flask and SQLite for managing event registrations.
